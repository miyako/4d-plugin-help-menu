Forcefully disable the Help menu; use with caution!

### Discussion

Cocoa code is used for v14 and 64 bits. Carbon code for older 4D.

---

## Syntax

```
HELP SET ENABLED (enabled)
enabled:=HELP Get enabled
```

Parameter|Type|Description
------------|------------|----
enabled|LONGINT|
